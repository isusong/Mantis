Detailed Description

Class for streaming vertices to the GPU using a buffer object. This speeds code performance and avoids the problem of OpenGL reserving upwards of 70% of the CPU memory for a buffer for glVertex4f commands.

You need to have a valid OpenGL context before creating a StreamBuffer. The StreamBuffer will piggyback on the current context.
