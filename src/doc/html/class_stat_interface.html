Detailed Description

Class that communicates with the statistics package to perform the statistics comparison.

Author
Laura Ekstrand, though large portions of compare were copied from PrintTrimmedOneOne in the stat package.

QVector< float > StatInterface::trimProfileEnds ( Profile *  data)
protected

Trim the masked ends off of a Profile and return it as a QVector<float>.

Masked points in the middle of the Profile will be included in the QVector<float>. This is because the stat package does not understand the concept of masked points.

Does not delete pointer.
